# moviesmatch

Эта игра в которой нужно угадать фильм по кадру. 

## Tech Stack
* Vue 3 Composition
* TypeScript
* Pinia
* Primevue
* Prettier
* EsLint
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
